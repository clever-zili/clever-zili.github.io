Cartridge

Clever / December 29, 2020

Cartridges is a puzzle game where you have to connect the circuit to finish a game cartridge. The game has 40 levels, each one representing the manufacturing of an old game.

In this game, you have to connect the circuit’s power source to all opened ends. Each piece in the level can be rotated to build this connection, but you need to hurry up before the time runs out.
